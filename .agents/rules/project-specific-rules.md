### ⚠️ Auto-Deployment Rule
**CRITICAL:** NEVER auto-deploy the application (e.g., using `wrangler deploy` or `wrangler d1 execute`) via terminal commands without the user's explicit request. 
Instead of running deployment or production database altering commands directly, **always provide the required command in the chat** so the user can review and run it manually.
